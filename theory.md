# Жизненный цикл ПО
- это ряд событий, происходящих с системой в процессе её создания и дальнейшего использования.

## Каскадная модель ЖЦПО

> Выработка системных требований
>> выработка требований к ПО
>>> анализ
>>>> проектирование
>>>>> кодирование
>>>>>> тестирование
>>>>>>> эксплуатация

## Инкрементная модель ЖЦПО

> анализ - проектирование - программирование - тестирование
>> анализ - проектирование - программирование - тестирование
>>> ...
>>>> анализ - проектирование - программирование - тестирование

## Итеративная модель ЖЦПО

> анализ требований - проектирование - программирование - тестирование
>>  анализ требований - проектирование - программирование - тестирование - развёртывание
>>>  проектирование - программирование - тестирование - развёртывание - эксплуатация



``` Agile = Инкрементная модель + Итеративная модель ```
